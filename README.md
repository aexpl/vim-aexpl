# AEXPL Vim Plugin

## Installation

Install with your favorite Vim plugin manager.
We recommend [vim-plug](https://github.com/junegunn/vim-plug).

- Add `Plug 'dummyunit/vim-fastbuild'` to your `.vimrc`
- Restart Vim
- Execute `:PlugInstall`
